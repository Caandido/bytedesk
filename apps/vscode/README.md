# ByteDesk para VSCode

Anote estudos e **salve códigos no ByteDesk sem sair do editor**.

## Recursos

- **Entrar** com sua conta do ByteDesk (e-mail + senha; o token fica no
  SecretStorage do VSCode).
- **Árvore de Estudos** na barra lateral — expanda um estudo para ver seus
  arquivos de código (clique para abrir num editor).
- **Salvar seleção como código**: selecione um trecho no editor → menu de
  contexto → _"ByteDesk: Salvar seleção como código no estudo"_. O nome do
  arquivo e a linguagem são detectados automaticamente.
- **Novo estudo** e **abrir estudo no site**.

## Configuração

- `bytedesk.apiUrl` — URL base da API (padrão: produção).

## Uso

1. `ByteDesk: Entrar` (paleta de comandos) e escolha o workspace.
2. Selecione código e use _Salvar seleção como código no estudo_.
